Основы сетевых технологий
	IP-адрес это адрес устройства который нужен для того чтобы устройства могли обращаться друг к другу. 
	TSP нужен для передачи данных в которых важно передать данные без ошибок и потерь 
	UDP используется для передачи потоковых данных, таких как видеострим, где потеря некоторого количества данных не критически важна

Логическое мышление
	Включаем первый выключатель на несколько минут
	Включаем второй выключатель, а первый выключаем
	Заходим в комнату
	Лампочка которая сейчас горит относится ко второму выключателю
	Касаемся выключенных лампоче
	Холодная лампочка относится к третьему выключателю
	Горячая лампочка относится к первому выключателю
